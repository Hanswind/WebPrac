# 9 . 트랜지션(transitions)

: **이동/변경을 멋지게 보여주는 효과**

<br>

#### [트랜지션 기본구조]

```css
transition: property 초 효과;

/* 예시 */
.box:hover {
	transition : background-color 4s ease-in-out
}
```

: 트랜지션은 **focus. active, hover** state 들에 대해서 효과적으로 적용 된다.
