#### [웹페이지에 자바스크립트 적용방법]

1. **웹페이지에 자바스크립트 작성**

   ```html
   <!DOCTYPE html>
   <html>
       <head>
           <meta charset='utf-8'
       </head>
       <body>
           ...
           <script>
           	// 자바스크립트 내용 작성
           </script>
       </body>
   </html>
   ```

2. **웹페이지에 자바스크립트 파일 적용**

   ```html
   <script src="main.js"></script>
   ```
