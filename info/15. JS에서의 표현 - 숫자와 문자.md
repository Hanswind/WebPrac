# 15. JS에서의 표현 - 숫자와 문자

#### [숫자]

: 큰따옴표나 작은따옴표가 붙지않은 숫자는 숫자로 인식.

1. **사칙연산**

   ```javascript
   alert(1 + 1);
   alert(2 * 5);
   alert(6 / 3);
   alert(5 % 3);
   ```

2. **Math 함수**

   ```javascript
   // 1. 제곱승
   Math.pow(3, 2);      // 3의 2승 = 9
   
   // 2. 반올림
   Math.round(10.6);    // 11
   
   // 3. 올림
   Math.ceil(10.1);     // 11
   
   // 4. 내림
   Math.floor(10.7);    // 10
   
   // 5. 제곱근
   Math.sqrt(9);        // 3
   
   // 6. 랜덤숫자
   Math.random();      // 0 ~ 1.0 사이의 랜덤한 숫자
   ```

3. **기타 숫자 함수** : https://opentutorials.org/course/50/39

#### [문자]

: 문자는 "" or '' 중 하나로 감싼다.

 1. **문자 기본 형태**

    ```javascript
    alert("Hello");
    alert('Hello');
    
    alert("'Hello' is String!");  // 'Hello' is String
    alert('Hello\' is String');   // Hello' is String
    
    alert("Hello\n is String")    // 여러줄의 표시
    
    alert("Hello" + " World!");   // 문자열의 결합
    
    alert("Hello".length);        // 5
    ```
